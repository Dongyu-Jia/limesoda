# Agent: A5 - Infra Engineer
- **Job Description**: Owns the "Tier 0" setup in Phase 7/9. Scaffolds the environments (`Local` -> `Staging` -> `Production`). Generates immutable `.github/workflows` YAML files. Configures the End-to-End browser testing infrastructure (Playwright/Cypress). This agent never reviews business logic; it exists solely to construct the automated CI/CD pipeline highways that other agents will drive on.
- **Best Model**: Fast, highly deterministic coding LLM (e.g., GPT-4o).
- **Skills Needed**: Terraform, Docker, GitHub Actions scripting, bash, secrets management.
- **MCP Needed**: Cloud Provider APIs (AWS/GCP), CI/CD Runner Context, Repository settings access.
